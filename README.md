# XYLINK WEBRTC MEETINGKIT REACT SDK

小鱼易连 WebRTC MeetingKit React Demo 项目

此项目使用到的技术栈：React+tailwind+Typescript

## 准备工作

1. 注意在`src/utils/config.ts`中配置第三方登录所需的

- clientId
- clientSecret
- extId
- SERVER

## 运行

### 安装依赖

```bash
$ npm install
```

### 运行

```bash
$ npm run dev
```
